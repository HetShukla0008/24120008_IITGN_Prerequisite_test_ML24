Q1¶

Total multiplications = n x m x k Total additions = n x m x (k-1)

In matrix multiplication of (n x k) and (k x m) matrices, each element of each row of matrix-1 is multiplied with each corresponding element of each column of matrix-2. Thus for multiplication of a single row having k elements with single column having k elements has total k multiplications.These k multiplications are added to form a single element in product matrix so there are corresponding k-1 additions here. Now there are m columns in matrix-2, so each row is further multiplied m times, each having k multiplications and k-1 additions. So the number of multiplications and additions for each row of matrix-1 is m x (k) and m x (k-1) respectively. Further, there are n rows so total multiplications will become nmk and total additions will be n x m x (k-1).

import numpy as np

def median_m1(list1,n):
    l = list1.copy()
    record = [0]*n
    max_el=float(l[0])
    max_idx=0
    k=0
    count=0
    while ((n%2==0 and count <= (n/2)) or (n%2!=0 and count <= (n+1)/2)):
        for i in range(k,n):
            if record[max_idx]!=1 and float(l[i])>= max_el:
                max_el=float(l[i])
                max_idx=i
        record[i] = 1                
        count+=1
        k+=1
    mid=max_idx
    if n%2==0:
        return (float((l[int(mid/2)]+l[int(mid/2 + 1)])/2))
    else:
        return (float(l[int((mid+1)/2)]))

Q12¶

Broadcasting is a way in which two numpy arrays of different dimensions can be operated upon by various arithmetic operations like addition, multiplication and so on. The array which lags in dimensions is stretched or broadcasted in other directions so that they can be operated just like two same dimensional arrays.

In [18]:
# CODE :

import numpy as np

m1=np.ones((2,3))

m2=np.arange(3)

# m2 lags behind in dimensions so left end is given 1 row.

# m1 (2,3) and m2 (1,3) 

# Now m2 has 1st dimension mismatched with m1 so it is stretched or broadcasted to form same dimension. therefore now  m1(2x3) =[[1,1,1,],[1,1,1]]
# and m2(2x3) =[[0,1,2],[0,1,2]]. If we add both of them we get [[1,2,3],[1,2,3]], a 2x3 matrix.

print(m1+m2)

 [1. 2. 3.]]

import numpy as np

def min_idx_calc(arr):
    min_val = arr[0]
    min_idx=0

# Minimum element's index without using argmin:

    for i in range(len(arr)):
        if arr[i]<min_val:
            min_val=arr[i]
            min_idx=i
            
    return min_idx

arr = np.array([1,3,-2,4,0,-3])

print("Minimum element's index without using argmin: "+str(min_idx_calc(arr)))

print("Minimum element's index using argmin : "+ str(np.argmin(arr)))
